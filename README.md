# Calculadora de terminal

Esta calculadora de terminal nos ayuda a realizar operaciones básicas y guarda un historial de las operaciones realizadas.

## Motivación del proyecto

Este proyecto ha sido creado por el profesor Jordi Cidoncha para trabajar los conceptos de documentación con sus alumnos de los ciclos de DAM/DAW en el ámbito del módulo Entornos de Desarrollo.

## Tecnologías usadas

Este proyecto está desarrollo al 100% con Java

## Como usar el proyecto

Para poder usar el proyecto debes:

1. Clonar el repositorio:

```bash
git clone https://github.com/jordicido/calculadora-doc
```

2. Antes de poder ejecutar el proyecto, asegúrate de tener instalado [JDK](https://www.oracle.com/es/java/technologies/javase/jdk11-archive-downloads.html)
3. Abrir el proyecto con tu IDE favorito
4. Ejecutar el archivo Main.java para empezar a utilizar la calculadora de terminal
